Send-SeKeys
  Element and keys parameters are both mandatory

   Function Renamed to validateUrl (private helper function does not need to use verb-noun naming and "validate" verb upsets script analyzer)
   added old name backs as an alias (which _is_ allowed. )


New-SeScreenShot alias SeScreenshot . Merges Invoke-ScreenShot and SaveSeScreenShot
   Can pass -path,
            -passthru (Alias PT)
            -path & -passthru
       or   -AsBase64
   "Target" used instead of "driver", with alias "Driver", and will come from $Global:SeDriver
   and throws an error if no driver passed or found from Global var


Start-Fe-Firefox & Start-SeChrome
  User can now write in a sort or DSL CHROME $URL etc.
  Trap failure to return driver, raise a warning
